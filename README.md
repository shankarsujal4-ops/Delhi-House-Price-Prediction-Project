# Delhi-House-Price-Prediction-Project
End-to-end machine learning project to predict Delhi house prices using features like area, geographical location, BHK, bathroom, balcony and parking.

##  Features Used
- Area
- Geographical Location (latitude, longitude)
- Number of Bedrooms (BHK)
- Number of Bathrooms
- Number of balcony
- Number of parking slots
- Furnish Status
- resale_status

##  Model
- Linear Regression
- Lasso
- Ridge
- Random Forest Regression

##  Results
Achieved good prediction performance on test data, and compared r2 scores of all the models

##  Tools Used
- Python
- Pandas
- Scikit-learn
- Google Colab
